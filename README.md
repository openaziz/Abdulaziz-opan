#Abdulaziz AI Platform

منصة ذكاء اصطناعي متكاملة تم تطويرها بواسطة **Abdulaziz**، تجمع بين تقنيات حديثة في التعامل مع الـ APIs وواجهات برمجية مخصصة، مع دعم كامل لنماذج LLM وتوليد الاستجابات الذكية.

## المميزات

- دعم كامل لتشغيل نماذج الذكاء الاصطناعي محليًا وعبر الويب.
- تكامل مع Git LFS لإدارة ملفات النماذج الثقيلة.
- واجهة برمجية منظمة ومرنة باستخدام Python.
- دعم واجهات الاستخدام المستقبلية (frontend integration ready).
- بنية مشروع نظيفة وسهلة التوسعة.

## التركيب والتشغيل

```bash
git clone https://github.com/openaziz/Abdulaziz-opan.git
cd Abdulaziz-opan
pip install -r requirements.txt
python app.py

ملاحظات مهمة

تأكد من إعداد بيئة Python 3.10+.

لتشغيل النماذج المحلية: تأكد من تحميل الملفات عبر Git LFS.

لا توجد مفاتيح API مضمّنة داخل الكود لضمان الأمان.

هذا المشروع مخصص لتجارب Abdulaziz الخاصة في بناء المنصات الذكية.


الهيكل العام للمشروع

Abdulaziz-opan/
│
├── app.py                   # نقطة التشغيل الأساسية
